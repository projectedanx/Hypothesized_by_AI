# ADR 002: Relational Sovereignty vs Extractive Sprints

## Status
Accepted

## Context
Continuous deployment rituals and "Follow-the-Sun" global delivery workflows often serve as mechanisms for the extraction of intellectual labor, prioritizing velocity over sustainability. Time-tracking software and burndown charts enforce able-bodied normativity (Crip-Time Genealogy), marginalizing neurodivergent and non-linear cognitive rhythms.

## Decision
We adopt the Relational Sovereignty framework. The paradigm shifts from viewing developers as resources to be leveled, to treating them as participants within a relational ecosystem. Project milestones must prioritize the health of the systemic network over rigid deadline completion.

## Consequences
- Operational redesigns will accommodate diverse cognitive rhythms.
- Velocity metrics are subordinated to relational network health.
- Empirical documentation updates are governed by these non-obvious analytical lenses, valuing researcher subjectivity and holistic system integration.
